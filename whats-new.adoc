== 2020 年 12 月 7 日



=== Cloud Manager と Spot の間のナビゲーション

Cloud Manager と Spot の間の移動が簡単になりました。

Spot の新しい「 * ストレージ運用 * 」セクションでは、 Cloud Manager に直接移動できます。作業が完了したら、 Cloud Manager の * Compute * タブから Spot に戻ることができます。



== 2020 年 10 月 18 日
